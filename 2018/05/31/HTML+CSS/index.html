基础点
  • Study

新发展

html是内容，css是表现，javascript是行为。Html 是最基础的，现在流行的是html5，先学会网页布局。HTML5是由HTML演化而来的。css是美化HTML页面的，为页面提供布局和格式。

精灵技术

将一个大图片里面的一部分在网页里面显示，局部显示。设置宽度和高度之后，用定位的方法来实现。

难点代码

1、ctrl+f 查找；ctrl+h 查找并替换
2、id在css里面是#，class在css里面是”.”。
3、cursor: pointer; /鼠标调成小手/
4、white-space: nowrap; /强制不换行/
5、
overflow: hidden; /清除浮动、溢出隐藏/
height: 20px; /隐藏时要加上高度才有效果/
margin-left: -1px; /配合溢出隐藏将左边框去掉/
5、
border-top-right-radius: 5px;将右上角的矩形变成圆形。
border-radius: 0px; /去除圆角/
border-top-left-radius: 0px; 去除左上角圆角。
z-index: 300; /设置级数、优先级，可将优先级低的遮到下面不显示/
5、
用select进行下拉列表的设置，用option对下拉列表进行填充值。
6、vertical-align: top;让文字垂直居中。
7、class=”fn-hide” ： 隐藏该类的内容。
8、”>”表示”>”
9、W3school 手册。
10、”¥”表示金钱符号：”￥”。”©” copy符号.
11、当一个大页面中，要求做出来的网页都在屏幕的中间是用：margin: 0 auto; /居中/。
